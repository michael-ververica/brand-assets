# brand-assets

Public Ververica brand asset library.

## What’s included

- Logos (RGB)
- Icons
- Illustrations
- Patterns
- Trust seals

## What’s not included

- Strategy, writing guidance, prompts, or internal design documentation
- CMYK exports

## Use via raw URLs

Base:

- https://raw.githubusercontent.com/michael-ververica/brand-assets/main/

Example:

- https://raw.githubusercontent.com/michael-ververica/brand-assets/main/assets/patterns/Glitch%20Pattern/pattern%20white.svg

## Asset index

- `manifest/assets.json` contains a machine-readable list of files and paths.
